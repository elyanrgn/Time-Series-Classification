# Config Files

This folder stores YAML configurations used for transfer experiments.

- `backbone.yml`: IndPatchTST backbone config from ETTh1 pretraining.
- `scratch_backbone.yml`: backbone config for scratch training on LSST.
- `scratch_train.yml`: training hyperparameters for scratch strategy.
- `params_head_only.yml`: Optuna best params for head-only fine-tuning.
- `params_late_enc.yml`: Optuna best params for late-encoder fine-tuning.
- `params_full_tune.yml`: Optuna best params for full fine-tuning.

These files are auto-generated by running:
`python -m src.models.indpatchtst.py `
`python -m src.training.adapting_to_classification`
